# 🐺 WolfBubble

WolfBubble is a simple bubble shooter game built with **HTML5 & JavaScript**.  
Shoot and match bubbles to score points!

---

## 🎮 Features
- Classic bubble shooter gameplay  
- Clean UI/UX  
- Responsive design  
- Lightweight and fast
